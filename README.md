# gse2srr.sh: convert GSE ID to SRR

## Features

- minimal dependencies (only `bash` and `wget`)

## Usage

```bash
./gse2srr.sh <GSE Number>
```

### Example

```bash
./gse2srr.sh GSE199916
# SRR18579294
# SRR18579296
# SRR18579292
# SRR18579291
# SRR18579290
# SRR18579293
# SRR18579297
# SRR18579295
```
